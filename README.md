 Aurelia by Her - Hotel Booking System

Responsive hotel booking website for boutique hotels in Kenya. Browse rooms, view KES pricing, and book directly via WhatsApp.

Live Demo: https://aureliabyitzher.netlify.app/
Tech Stack: HTML, CSS, JavaScript

 Features
- 3 Room Types: Single KES 8,500, Double KES 12,000, Suite KES 18,500 per night
- Direct WhatsApp Booking: "Book Now on WhatsApp" buttons for instant customer contact
- Clean, mobile-responsive design for all devices
- Room Amenities: WiFi, ensuite bathrooms, balcony, minibar, complimentary breakfast, separate lounge
- Elegant UI focused on user experience and comfort
